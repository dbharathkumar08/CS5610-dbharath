Introduction

Its Movie Time is my final project for CS5610 course. Its a Movie database that helps the user explore various movies, and make his own diary of movies. It also heps him to explore other users and folllow them if he likes their movie choices.

List of available functionalities

Functionality Use
Home Page This is my landing page. users have the login/Registration Buttton. Search bar to search for the required movie They also have the option to see all the other users
Login/Signup Page It is a modal through which user can login or Sign up
Search Page Users can search the movies using this search bar
Description Page User can see the brief details of all the movies searched based on this title.
Details Page Users can see the details of the selected movie.Exposed details- All the people liking the movie, all the reviews and ratings of the movie
Profile Page Page This page gives all the details of the current logged in user.It gives the details of all the movies the user has liked. his reviews on the movie.It will also give details about the movies of his favorite
Other User Profile In this Page user can see the details of other users who have registered

Technologies Used

TECHNOLOGY PURPOSE USED IN
HTML5 All pages are designed using HTML5 Front end/Client Side
CSS / Bootstarp This is use to enhance the look and feel of project and bootstarp also makes it responsive Front end/Client Side
jQuery This is technology handles interlinking of elements and for calling webservices that provide data to us. Client Side
Java Script JS is used to handle some onclick functionalities. Client Side
Bootstrap-UI Used for modal services Client Side
Angular.JS It is used to handle data from webservices and provide single page look to the application Client Side
Node.js This technology is the backbone of server side functionality. Server Side
Express (Node.js Module) This helps in establishing web services on which the project work and various services like body parsing etc Server Side
Passport (Node.js Module) This helps in user authentication and validation. Server Side
MongoDB This is the backend used for data storage purpose Data Storage
Mongoose Helps in integrating MongoDB at server side Server Side
