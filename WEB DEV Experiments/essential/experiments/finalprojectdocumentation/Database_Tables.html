Data Storage - MongoDB

Besides data provided from APIs we have to maintain user data locally,so for backend storage purpose I am using MongoDB which is a no SQL database. In mongoDB there are collections instead of tables and documents (JSON Objects) instead of tupples.
The Coolections for my project are :

  • UserModel : usermodels
    • Field Type Usage
    username String user's unique username
    password String User's password
    email String User's mail id
    Followerusers Array Username of those whom the user follows
    Followingusers Array User name of those who follow the user
    userfavmovies Array IMDB id of the movies which the user has made fav
    firstName String User's first name
    lastName String User's last name
    address String User's address
  • MovieModel: moviemodels
    • Field Type Usage
    idIMDB String IMDB id of the movie for which review is done
    plot String Plot of the Movie
    rating String The rating of the movie given by critics
    title String Movie title
    year String The year the movie was released
    username Array List of users who have this movie as their favorite
  • ReviewModel: reviewmodels
    • Field Type Usage
    idIMDB String IMDB id of the movie for which review is done
    username String Unique username
    title String Title of the movie
    rating String rating given to movie by user
    comment String comment made on movie by user
