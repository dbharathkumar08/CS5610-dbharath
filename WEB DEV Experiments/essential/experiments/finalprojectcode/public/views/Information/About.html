Entertainment

        This project is about the way users can search the movies of their choice, and also explore other users.
        Features include:
        1)Login/Registration
        2)Search Movies- Fetches Data from IMDB Web Apis(Courtesy:myapifilms.com)
        3)See the movie details:
            a)Jump to IMDB links.
            b)Favorite the movie.
            c)Review the movie.
        4)Search for other users.
            a)follow/Unfollow the user.
            b)Read others review.
        This project has been developed on MEAN Stack.
            
        Enjoy the web experience.
