Description

This experiments shows how the UI can be populated by angular JS, which is used as a controller to display the model on the view - HTML

Code Snippet

Code Shows the MVC Behaviour of Angular JS

               <div ng-controller="HelloWorldController">
        <h1>Hello World Angular Application</h1>
        <h2>Contents Displayed ina table</h2>
        <table class="table table-striped">
            <tr ng-repeat="e in employees">
                <td>{{e.first}}</td>
                <td>{{e.last}}</td>
                <td>{{e.salary}}</td>
                <td>{{e.Location}}</td>
            </tr>
        </table>
        <h2>Displaying a single content</h2>
        <p>
            {{employee.first}}
            {{employee.last}}
        </p>      
        <br />  
    </div>
