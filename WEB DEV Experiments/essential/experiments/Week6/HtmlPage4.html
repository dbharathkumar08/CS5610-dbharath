Description

This experiment on the button click of reset, clears the form. On COPY TO MASTER, makes the user a master. ON COPY FROM MASTER, copies the values from master to user.

Code Snippet

This experiment makes use of the functions to keep a temporary copy of the fields. 

    angular.module('myApp', []).controller('formController', ['$scope', function ($scope) {
    $scope.master = { firstName: "Bhagat", lastName: "Singh" };
    $scope.reset = function () {
        $scope.user = { firstName: "", lastName: "" };;
    };
   
    $scope.copymaster = function () {
        $scope.master = angular.copy($scope.user);
    };
   
    $scope.copyfrommaster = function () {
        $scope.user = angular.copy($scope.master);
    };
   
}]);
