This Document gets hidden or shown on button click
This Document is used to demonstrate toggle button click

Description

Jquery show,hide and toggle functions are experimented here

Code Snippet

Code Below will show hide and toggle the document

        $(document).ready(function () {
    $("#toggle").click(function () {
        $("#two").toggle();
    });
    $("#hide").click(function () {
        $("#one").hide();
    });
    $("#show").click(function () {
        $("#one").show();
    });
});
