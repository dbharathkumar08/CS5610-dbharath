Movie Database

Descrption

This experiment involves usage of APIs that is used by my project inorder to fetch data . Here I fetch all the categories that can be fetched from the WEB API. I try to fetch all the movies in a particular genre and display its POSTER upon the click. and also the movie details. The display is ina broad spectrum. When the poster is clicked, Youtube video is also played.

Code Snippet

Code Below is the angular JS code that acts as a controller

     app.directive('hoverBgImage', function () {
    return {
        link: function (scope, elm, attrs) {
            elm.bind('mouseenter', function () {
                this.style.backgroundImage = 'url(' + attrs.hoverBgImage + ')';
            });
            elm.bind('mouseleave', function () {
                this.style.backgroundImage = '';
            })
        }
    };
});
